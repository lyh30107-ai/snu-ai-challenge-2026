# SNU AI Challenge 2026

This repository contains my experiments for the SNU AI Challenge.

## Task

Given a sentence and 4 shuffled video frames, predict the correct chronological order.

## Current Approach

- CLIP image/text encoder
- 24-class ordering classifier
- Validation split from train.csv
- Submission generation for Kaggle

## Notes

Competition data and model checkpoints are not included in this repository.
